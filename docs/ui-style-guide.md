# UI Style Guide

## Color Palette

### Primary (Solid Pod Purples)
- `#5B21B6` - Deep purple
- `#7C3AED` - Medium purple
- `#C084FC` - Light purple

### Accent
- `#14B8A6` - Teal

### Neutrals (Slate/Blue-gray)
- `#0F172A` - Darkest background
- `#1E293B` - Dark background
- `#334155` - Medium background
- `rgba(148,163,184,0.4)` - Translucent borders

## Glass Effects

- Blur: 16–18 px
- Alpha: 0.6–0.7
- Shadow: `0 25px 65px rgba(15,23,42,0.45)`
- Border radius: 12–16 px

## Typography

- Font family: Inter (or system sans fallback)
- Heading weight: 600
- Body weight: 400
- Line height: ≥ 1.5

## Motion

- Transitions: 200–240 ms ease-in-out
- Hover fades: 120–150 ms
- Easing: `cubic-bezier(0.4, 0, 0.2, 1)`

## Gradient Usage

Purple gradients (`#5B21B6` → `#C084FC`) layered on dark glassmorphic backgrounds.
