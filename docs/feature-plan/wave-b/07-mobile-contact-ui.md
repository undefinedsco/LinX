## 7. 交互设计规格（Interaction Design Review）

> 本节定义 Mobile Contact UI 中群组管理的移动端适配规格。

### 7.1 移动端群组创建

#### 入口

- 联系人 Tab → 右上角 "+" → "创建群组"
- 或从聊天 Tab → 右上角 "+" → "新建群聊"

#### 创建流程（多步骤页面）

移动端屏幕有限，群组创建拆分为两步：

**Step 1: 选择成员**
```
┌─────────────────────────────────────┐
│ ← 选择成员                   [下一步] │
├─────────────────────────────────────┤
│ 🔍 搜索联系人...                     │
├─────────────────────────────────────┤
│ 已选 (2): [Alice ✕] [Bob ✕]        │
├─────────────────────────────────────┤
│ 联系人                               │
│ ┌─────────────────────────────────┐ │
│ │ ☑ Alice                         │ │
│ │ ☑ Bob                           │ │
│ │ ☐ Carol                         │ │
│ └─────────────────────────────────┘ │
│ AI 助手                              │
│ ┌─────────────────────────────────┐ │
│ │ ☐ Claude 3.5 Sonnet             │ │
│ │ ☐ GPT-4o                        │ │
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

**Step 2: 群组信息**
```
┌─────────────────────────────────────┐
│ ← 群组信息                   [创建]  │
├─────────────────────────────────────┤
│                                      │
│        [📷 群头像]                   │
│                                      │
│  群名称                              │
│  [项目讨论组____________]            │
│                                      │
│  成员 (3人)                          │
│  [Alice] [Bob] [Claude]             │
│                                      │
└─────────────────────────────────────┘
```

### 7.2 移动端群成员管理

#### 群成员页面（push navigation）

```
┌─────────────────────────────────────┐
│ ← 群成员 (5)              [+ 邀请]  │
├─────────────────────────────────────┤
│                                      │
│ [头像] Alice              群主       │
│ [头像] Bob                管理员     │
│ [头像] Carol              ⚪ 离线    │
│ [🤖]  Claude              AI 助手   │
│ [头像] 你                            │
│                                      │
└─────────────────────────────────────┘
```

- 点击成员行 → 进入联系人详情页
- 长按成员行 → 弹出操作 sheet（查看资料 / @提及 / 移除）
- 移除操作需要二次确认（底部 ActionSheet）

#### 邀请成员

- 点击 "+ 邀请" → push 到联系人选择页（复用 Step 1 的选择器）
- 已在群内的联系人显示为灰色不可选

### 7.3 联系人列表移动端适配

- 分组显示：个人联系人 / AI 助手 / 群组，使用 section header 分隔
- 群组项：多头像拼接 + 成员数 + 最后活跃时间
- 滑动操作：左滑显示"删除"（个人）或"退出群聊"（群组）
- 搜索：顶部搜索栏，支持按名称和 WebID 搜索
